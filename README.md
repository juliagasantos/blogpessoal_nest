## 📌 Sobre o projeto

Este projeto consiste em uma API REST desenvolvida com **NestJS**, com o objetivo de gerenciar postagens de um blog pessoal.

A aplicação segue boas práticas de desenvolvimento backend, utilizando arquitetura modular, integração com banco de dados e organização de código profissional.

---

## 🚀 Tecnologias utilizadas

- NestJS  
- TypeScript  
- TypeORM  
- MySQL  
- Insomnia  

---

## ⚙️ Funcionalidades

✔️ Listar todas as postagens  
✔️ Integração com banco de dados  
✔️ Estrutura modular com NestJS
